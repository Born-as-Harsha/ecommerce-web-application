# E-Commerce Web Application

## Overview

A full-stack E-Commerce web application developed using Flask and SQLite that supports product management, shopping cart functionality, user authentication, role-based access control, and order checkout.

## Features

* User Authentication and Authorization
* Role-Based Access (Admin / User)
* Product Catalog Management
* Add to Cart Functionality
* Checkout System
* Admin Product Management Panel
* Database Integration using SQLite
* Responsive Bootstrap UI
* Dynamic Data Handling

## Tech Stack

**Frontend**

* HTML
* CSS
* Bootstrap

**Backend**

* Flask (Python)

**Database**

* SQLite
* SQLAlchemy ORM

## Installation

Clone Repository:

```bash
git clone <repository-url>
cd ecommerce-web-application
```

Install Dependencies:

```bash
pip install -r requirements.txt
```

Run Application:

```bash
python app.py
```

Open Browser:

```text
http://127.0.0.1:5000
```
